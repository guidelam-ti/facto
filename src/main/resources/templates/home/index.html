Étape A — squelette
Le projet est en place : Spring Boot 4, Thymeleaf, Bootstrap local, H2 fichier. Les écrans fonctionnels (Configuration, Scan, Fournisseurs, Traitement, Historique) arrivent dans les étapes suivantes.
1. Configuration

Connecter ton compte Google (Gmail + Drive) via OAuth 2.0 et choisir le dossier Drive racine d'archivage.

Configurer
2. Scan

Inspecter ta boîte de réception et regrouper les emails avec pièces jointes PDF par expéditeur.

Lancer un scan
3. Fournisseurs

Associer chaque adresse expéditeur à un fournisseur canonique (ou marquer comme à ignorer).

Mapper les expéditeurs
4. Traitement

Choisir une période, télécharger les PDF, les renommer et les uploader dans Drive organisés par année/mois.

Disponible à l'étape F
5. Historique

Consulter les factures déjà traitées avec filtres par année et fournisseur.

Disponible à l'étape G
